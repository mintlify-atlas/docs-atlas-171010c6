# Market Research Toolkit

The Market Research toolkit combines powerful website auditing with marketing psychology frameworks to help you understand, optimize, and improve digital experiences.

## What is the Market Research Toolkit?

This toolkit provides AI agents with two specialized capabilities:

1. **Website Auditing**: Comprehensive technical and marketing analysis powered by the squirrelscan CLI
2. **Marketing Psychology**: Application of 70+ mental models and behavioral science principles to marketing decisions

Together, these capabilities enable deep analysis of websites from both technical and psychological perspectives, providing actionable insights for improvement.

- **Actionable recommendations** for fixing issues

## Core Components

### 1. squirrelscan CLI

The `squirrel` command-line tool provides the auditing engine:

- **Cross-platform**: Available for macOS, Windows, and Linux
- **Browser emulation**: Renders pages like real browsers and search crawlers
- **Local database**: Caches results for fast re-analysis and regression tracking
- **Flexible output**: Console, LLM-optimized, JSON, HTML, Markdown, and more
- **Configurable coverage**: Quick, surface, or full audits depending on your needs

<Info>
The CLI is designed to work seamlessly with AI agents, providing structured, token-efficient output formats optimized for LLM consumption.
</Info>

### 2. AI Agent Skills

Two specialized agent skills provide high-level capabilities:

**audit-website skill**: Handles the complete audit workflow:
- Installation and setup of squirrel CLI
- Running audits with appropriate coverage modes
- Interpreting results and prioritizing fixes
- Parallelizing fixes across multiple agents for speed
- Iterating until health scores reach target levels (typically 85-95+)
- Regression testing to detect new issues

**marketing-psychology skill**: Applies mental models to marketing:
- Identifies relevant psychological principles for specific situations
- Explains the science behind buyer behavior
- Provides concrete marketing applications
- Ensures ethical implementation of persuasion techniques
- Tailors recommendations to product context when available

### 3. Mental Models Library

The toolkit includes 70+ mental models organized into categories:

- **Foundational Thinking Models**: Strategy, problem-solving, and decision frameworks
- **Understanding Buyers**: Cognitive biases and behavioral patterns
- **Influencing Behavior**: Ethical persuasion and behavior change
- **Pricing Psychology**: How people perceive and respond to prices
- **Design & Delivery**: Creating effective marketing systems
- **Growth & Scaling**: Models for sustainable, compounding growth

## How It Works Together

The power of this toolkit comes from combining technical auditing with psychological insights:

1. **Audit** a website to identify technical issues, broken user flows, and optimization opportunities
2. **Apply psychology** to understand why issues matter and how to fix them effectively
3. **Implement fixes** guided by both technical best practices and behavioral science
4. **Re-audit** to verify improvements and catch any regressions
5. **Iterate** until the site achieves optimal health and conversion performance

<Note>
The toolkit emphasizes **ethical application** of psychological principles. All persuasion techniques should be used to genuinely help customers make better decisions, not to manipulate or deceive.
</Note>

## Example Workflow

Here's how the toolkit might be used to improve a landing page:

```
1. Run website audit
   → Discovers: slow load time, missing H1, weak meta description, 
                no social proof, broken mobile layout

2. Apply psychology frameworks
   → Identifies: High activation energy (friction), no social proof,
                missing scarcity/urgency, unclear value proposition

3. Implement fixes
   → Technical: Fix mobile CSS, optimize images, add H1, expand meta
   → Psychological: Add testimonials (social proof), simplify form (reduce friction),
                   add "limited spots" (scarcity), clarify outcome (Jobs to Be Done)

4. Re-audit
   → Score improved from 68 (D) to 92 (A)
   → All critical issues resolved

5. Monitor conversions
   → Track whether combined technical + psychological improvements 
     drive better results
```
